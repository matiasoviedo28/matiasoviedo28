### Proyectos Destacados

#### [bomberos.ar](https://bomberos.ar)
### Fundador

Plataforma pensada para modernizar la gestión de cuarteles de bomberos voluntarios.

**Características**
- Control horario con carga automática al registro único nacional (RUBA)
- Múltiples perfiles con login para gestión de datos personales
- Legajos virtuales
- Manejo de múltiples asociaciones a nivel base de datos

#### [fwi.com.ar](https://fwi.com.ar)

Herramienta para monitoreo meteorológico y análisis predictivo.

- Desarrollo en Python para prevención en tiempo real de incendios forestales
- Uso de bases de datos SQL y análisis de archivos NetCDF
- Implementación en servidor dedicado con Ubuntu Server

#### Sistema de Alertas

Optimización de tiempos de respuesta en redes.

- Sistema de alertas en tiempo real para nodos y centrales caídas
- Enfocado en reducir tiempos de reparación
- Desarrollado con Python, SQL y servidores dedicados

#### Análisis de Demanda - Nordsee

- Modelo de Machine Learning para predecir demanda de productos
- Dashboard interactivo para visualizar insights clave
- Trabajo orientado a apoyar estrategia de expansión
